> **⚠️ Disclaimer:** This project is entirely a personal graduation gift. It has no official affiliation with any school, institution, or organization. The photos and content on the site are used solely for the purpose of commemorating graduation memories. If there is anything you would like removed, please feel free to contact me.

## 📋 About

This website is a personal project developed to celebrate a friend's high school graduation. It is not an official project; it was not created at the request or with the approval of any school or institution. I simply wanted to gather beautiful memories together and leave a small digital keepsake for classmates.

## <a id="features"></a>✨ Features

- **Video Player**: Watch the graduation video directly on the site via an embedded YouTube video
- **Photo Gallery**: Auto-sizing photo gallery in a bento grid layout
- **Lightbox**: Click to enlarge photos with keyboard arrow keys and button navigation support
- **Responsive Design**: Flawless appearance on mobile, tablet, and desktop devices
- **Modern UI**: Glassmorphism effects, smooth gradients, and hover animations
- **Auto Gallery** : Just add photos to the `public/photo-gallery/` folder; no code changes required

## 🛠️ Technologies <a id="technologies"></a>

- **Next.js**: Modern React framework using App Router and Server Components for optimal performance.
- **React 19**: Core UI library featuring concurrent rendering and server actions.
- **Tailwind CSS**: Utility-first CSS framework for fast and modern responsive design.
- **TypeScript**: Strongly typed programming language built on JavaScript for reliable and scalable development.

## 🚀 Installation <a id="installation"></a>

Follow the steps below to run the project in your local environment.

1. **Clone the Repository**

   ```bash
   git clone https://github.com/xkintaro/ttfl-graduation-memories.git
   cd ttfl-graduation-memories
   ```

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Start the Development Server**

   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser.
