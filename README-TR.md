> **⚠️ Sorumluluk Reddi:** Bu proje tamamen kişisel bir mezuniyet hediyesidir. Herhangi bir okul, kurum veya kuruluş ile resmi bir bağlantısı yoktur. Sitede yer alan fotoğraf ve içerikler, yalnızca mezuniyet anısı amacıyla kullanılmıştır. Kaldırılmasını istediğiniz bir şey olursa benimle iletişime geçebilirsiniz.

## 📋 Hakkında

Bu site, bir arkadaşımın lise mezuniyetini kutlamak amacıyla geliştirilmiş kişisel bir projedir. Resmi bir proje değildir; okulun veya herhangi bir kurumun talebi ya da onayı ile yapılmamıştır. Sadece güzel anıları bir araya getirmek ve küçük bir dijital hatıra bırakmak istedim.

## <a id="features"></a>✨ Özellikler

- **Video Oynatıcı**: Gömülü YouTube videosu ile mezuniyet videosunu doğrudan sitede izleme imkânı
- **Fotoğraf Galerisi**: Bento grid düzeninde, otomatik boyutlandırmalı fotoğraf galerisi
- **Lightbox**: Fotoğraflara tıklayarak büyütme, klavye ok tuşları ve butonlarla gezinme desteği
- **Responsive Tasarım**: Mobil, tablet ve masaüstü cihazlarda kusursuz görünüm
- **Modern Arayüz**: Glassmorphism efektleri, yumuşak gradyanlar ve hover animasyonları
- **Otomatik Galeri** : `public/photo-gallery/` klasörüne fotoğraf eklemek yeterli; kod değişikliği gerekmez

## 🛠️ Teknolojiler <a id="technologies"></a>

- **Next.js**: Optimum performans için App Router ve Server Components kullanan modern React framework'ü.
- **React 19**: Eşzamanlı render etme ve sunucu eylemlerini içeren temel kullanıcı arayüzü kütüphanesi.
- **Tailwind CSS**: Hızlı ve modern duyarlı tasarım için utility-first CSS framework'ü.
- **TypeScript**: Güvenilir ve ölçeklenebilir geliştirme için JavaScript üzerine inşa edilmiş strongly typed programlama dili.

## 🚀 Kurulum <a id="installation"></a>

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları takip edebilirsiniz.

1. **Depoyu Klonlayın**

   ```bash
   git clone https://github.com/xkintaro/ttfl-graduation-memories.git
   cd ttfl-graduation-memories
   ```

2. **Bağımlılıkları Yükleyin**

   ```bash
   npm install
   ```

3. **Geliştirme Sunucusunu Başlatın**

   ```bash
   npm run dev
   ```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini açın.
